# Evolution Simulator
Evolution simulation for the Object Oriented Programming course at AGH University 2022/23.

## Usage
```
git clone https://github.com/miwnek/Evolution-Simulator
cd Evolution-Simulator
./gradlew build
./gradlew run
```
